步骤 1：问题引入

考虑一般形式的一元二次方程：

ax2 + bx + c = 0（a ≠ 0）

目标：找到 x 的通解公式，即用系数 a, b, c 表达 x。

为什么需要通解公式？

对于任意给定的一元二次方程，如果每次都手工配平方，效率很低。通解公式使我们可以代入 a, b, c 直接计算结果。

步骤 2：移项与系数化一

将常数项移到等式右边：

ax2 + bx = -c

两边同时除以 a（因为 a ≠ 0）：

x2 + bax = -ca

现在方程变成了 x² + px = q 的形式，这是配平方的标准起点。

步骤 3：配平方

配方原理：x² + px 加上 (p/2)² 后成为完全平方 (x + p/2)²。

这里 p = b/a，所以需要在等式两边同时加上 (b/(2a))²：

x2 + bax + (b2a)2 = -ca + (b2a)2

左边可写成完全平方：

(x + b2a)2 = b2 - 4ac4a2

步骤 4：开方求解

对等式两边开平方：

x + b2a = ±√b2 - 4ac4a2

最终得到一元二次方程求根公式：

x = -b ± √(b2 - 4ac)2a

其中 Δ = b² - 4ac 称为判别式，决定根的性质：

  • Δ > 0 → 两个不等实根
  • Δ = 0 → 两个相等实根
  • Δ < 0 → 两个共轭复根

步骤 1：牛顿第二定律

牛顿第二定律描述了力与运动的关系：

F = ma

F — 合外力（N），m — 物体质量（kg），a — 加速度（m/s²）

这是经典力学的核心方程。今天我们从它出发，推导匀加速直线运动的全套公式。

步骤 2：从力到加速度

如果合外力 F 是常数，则加速度 a 也是常数：

a = Fm = const

加速度是速度对时间的导数：a = dv/dt。积分可得：

v(t) = v0 + at

其中 v₀ 是初始速度（t=0 时的速度）。

步骤 3：从速度到位移

速度是位移对时间的导数：v = dx/dt。对 v(t) 积分：

x(t) = 从 0 到 t 对 (v0 + aτ) 积分 = v0t + 12at2

如果初始位置为 x₀：

x(t) = x0 + v0t + 12at2

步骤 4：速度-位移关系

消去时间 t 可得到不依赖时间的速度-位移关系：

v2 = v02 + 2a(x - x0)

这就是匀加速直线运动的三公式：

v = v0 + atx = x0 + v0t + 12at2v2 = v02 + 2a(x - x0)

步骤 1：酸碱中和反应

盐酸与氢氧化钠的反应是最经典的酸碱中和：

HCl + NaOH → NaCl + H2O

酸（HCl 提供 H⁺）+ 碱（NaOH 提供 OH⁻）→ 盐 + 水。

步骤 2：离子方程视角

在溶液中，强电解质完全电离。写出离子方程式：

H+ + Cl- + Na+ + OH- → Na+ + Cl- + H2O

Na⁺ 和 Cl⁻ 在反应前后不变（旁观离子），消去后得到净离子方程式：

H+ + OH- → H2O

这揭示了酸碱中和的本质：H⁺ 与 OH⁻ 结合生成水。

步骤 3：氧化还原配平

以高锰酸钾氧化二价铁为例：

MnO4- + Fe2+ + H+ → Mn2+ + Fe3+ + H2O

还原半反应：

Fe2+ → Fe3+ + e-

步骤 1：什么是冒泡排序？

冒泡排序（Bubble Sort）是最简单的排序算法之一。其核心思想是：

相邻元素两两比较，逆序则交换。每一轮将当前最大的元素"浮"到最后。

步骤 2：第一轮演示

初始数组：[5, 3, 8, 1, 2]

第一轮结束后，最大值 8 已到达末尾位置。

步骤 3：完整过程

初始数组：[5, 3, 8, 1, 2]

共 4 轮完成排序。

步骤 4：复杂度分析

最坏时间复杂度：O(n2)　最好时间复杂度：O(n)　平均：O(n2)

空间复杂度 O(1)（原地排序）。冒泡排序简单直观，但对于大数据集效率低，通常用作教学示例。
